Cuestionario I

¿Qué es entonces una base de datos - DB?

Una base de datos se puede definir como un conjunto de información relacionada que se encuentra agrupada ó estructurada.Cada base de datos se compone de una o más tablas que guarda un conjunto de datos.

¿En qué casos es conveniente usar bases de datos?

Es conveniente el uso de una base de datos para cualquier empresa de manufactura, incluyendo información de ventas, compras, almacenes, producción, calidad, mantenimiento y finanzas. Sin embargo, el factor más importante a considerar es el volumen de los datos, especialmente si se trata de un gran número de registros.

¿Qué es un sistema de gestión de base de datos - DBMS?

Es una colección de datos interrelacionados y un conjuto de programas para acceder a estos. Se denomina "colección de datos" a una base de datos que contiene información de una organización.

Los sistemas de bases de datos (DBMS)estan diseñados para gestinar grandes bloques de información. El manejo de estos datos, comunmente requiere de estructuras de almacenamiento y mecanismos de gestión.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • Implantación de la integridad: Refiere a las restriciones a las que se ajusta una base de datos. Ex: El número de horas máximo de trabajo de un empleado es de 80 horas.
  • Implantación de la seguridad: Refiere a la jerarquía que debe tener una base de datos, puesto que todos los usuarios no pueden tener acceso al todo el contenido de la misma.
  • Copia de seguridad y recuperación: Refiere a la capacidad de detectar los fallos y restaurar una base datos.
  • Control de concurrencia: Refiere a que a pesar la base datos es modificada por varios usuarios, es nesesario que se matenga actualizada para que su contenido sea consistente.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En nuestro proyecto final de Diseño Interactivo desarrollamos una plataforma que permitía que niños aprendieran matemáticas desde sus casas. Esta página era manejada por un adulto en cuya cuenta se guardaba el progreso de cada estudiante. Pienso que aquí hubiera sido eficiente el uso de una base de datos puesto que por medio de ella hubiésemos podido registrar la información de cada padre de familia: nombre del padre, información del estudiante y progreso realizado.
